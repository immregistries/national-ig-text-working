Systems may choose to send messages in batches. A batch begins with a batch header statement (BHS) and ends with a Batch Trailer Segment. Batches may in turn be batched into files of batches using File Header Statement and File Trailer statement. If a system is sending a single batch, the FHS/FTS is not necessary. A stream of messages may be sent without use of either BHS or FHS.

The generic layout of a batch message is as follows:

BHS

VXU

VXU

BTS

Similarly, a file of batches is laid out as follows:

FHS

BHS

VXU

VXU

BTS

BHS

VXU

BTS

FTS
