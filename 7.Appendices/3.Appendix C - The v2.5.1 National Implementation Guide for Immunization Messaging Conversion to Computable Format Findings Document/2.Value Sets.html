The concepts of Extensibility, Stability, Content, and code usage information are not stated in the National IG. Additionally, two value sets, HL70064-Financial class and HL70636-Assigning authority, had local code placeholders, as indicated by three asterisks (***) or an ellipse (…), removed from their value sets. These values were removed to ensure that other NIST tooling would not literally interpret the local code placeholders as actual values.

While not specified in the National IG, value sets maybe defined as static or dynamic as this information was defined in IGAMT due its adherence to conformance methodology. Dynamic table data was updated according to the information available on the original sources' websites at the time of development.
